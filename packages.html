# pbdR Packages We maintain stable distributions of all of our packages on the [Comprehensive R Archive Network (CRAN)](http://r-project.org/). For information about installing these packages, see our comprehensive set of [installation instructions](install.html). Additionally, you can find current binaries of our packages for the Windows operating system at our sister site, [HPSC](http://thirteen-01.stat.iastate.edu/snoweye/pbdr/?item=download). ## Application | Package | Description | Download | |---------|------------- | :------: | | pmclust | Parallel model-based clustering. | [ghbutton](https://github.com/RBigData/pmclust) <br> [cranbutton](http://cran.r-project.org/package=pmclust) | pbdDEMO | Package demonstrations and examples. | [ghbutton](https://github.com/RBigData/pbdDEMO) <br> [cranbutton](http://cran.r-project.org/package=pbdDEMO) ## Computation | Package | Description | Download | |---------|------------- | :------: | | pbdDMAT | Distributed matrix methods. | [ghbutton](https://github.com/RBigData/pbdDMAT) <br> [cranbutton](http://cran.r-project.org/package=pbdDMAT) | pbdBASE | Base Wrappers for distributed matrices. | [ghbutton](https://github.com/RBigData/pbdBASE) <br> [cranbutton](http://cran.r-project.org/package=pbdBASE) | pbdSLAP | The Scalable Linear Algebra Package. | [ghbutton](https://github.com/RBigData/pbdSLAP) <br> [cranbutton](http://cran.r-project.org/package=pbdSLAP) | pbdMPI | Interface to MPI. | [ghbutton](https://github.com/RBigData/pbdMPI) <br> [cranbutton](http://cran.r-project.org/package=pbdMPI) ## I/O | Package | Description | Download | |---------|------------- | :------: | | pbdNCDF4 | Parallel NetCDF version 4. | [ghbutton](https://github.com/RBigData/pbdNCDF4) <br> [cranbutton](http://cran.r-project.org/package=pbdNCDF4) ## Profiling | Package | Description | Download | |---------|------------- | :------: | | pbdPROF | MPI profiling tools. The package offers utilities for linking with MPI-using R packages (pbdMPI and Rmpi), as well as tools for parsing profiler output files. fpmpi and mpiP are supported. | [ghbutton](https://github.com/RBigData/pbdPROF) <br> [cranbutton](http://cran.r-project.org/package=pbdPROF) | pbdPAPI | Bindings for PAPI. The package offers utilities for measuring and collecting performance counter data, such as number of floating point operations or cache misses. Linux (or FreeBSD with a custom kernel) only. | [ghbutton](https://github.com/RBigData/pbdPAPI) | scribe | Profiler visualizations. The package offers numerous plotters for data generated by pbdPROF and pbdPAPI. The package is kept separate so as not to include the graphics dependency chain on pbdPAPI and pbdPROF, as these dependencies may not be easily installed on supercomputers. | [ghbutton](https://github.com/RBigData/scribe) # Installation Instructions Below are sets of pbdR installation instructions for each major OS platform (or all three). These instructions include information for installing MPI and R in addition to installing the pbdR packages. If you have questions about installing a particular pbdR package, then you may need to consult with that package's vignette, which you can find on the [download page](download.html). We also maintain a Linux virtual machine (VM) with all of the pbdR packages installed and set up. You can find information about it [here](http://thirteen-01.stat.iastate.edu/snoweye/pbdr/?item=vm).
