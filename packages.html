# pbdR Packages We maintain stable distributions of all of our packages on the [Comprehensive R Archive Network (CRAN)](http://r-project.org/). For information about installing these packages, see our comprehensive set of [installation instructions](install.html). Additionally, you can find current binaries of our packages for the Windows operating system at our sister site, [HPSC](http://thirteen-01.stat.iastate.edu/snoweye/pbdr/?item=download). ## Application | Package | Description | |---------|------------- | | [pmclust](http://cran.r-project.org/package=pmclust) | Parallel model-based clustering. | [pbdDEMO](http://cran.r-project.org/package=pbdDEMO) | Package demonstrations and examples. ## Computation | Package | Description | |---------|------------- | | [pbdDMAT](http://cran.r-project.org/package=pbdDMAT) | Distributed Matrix Methods | [pbdBASE](http://cran.r-project.org/package=pbdBASE) | Base Wrappers for Distributed Matrices | [pbdSLAP](http://cran.r-project.org/package=pbdSLAP) | Scalable Linear Algebra Package | [pbdMPI](http://cran.r-project.org/package=pbdMPI) | Interface to MPI ## I/O | Package | Description | |---------|------------- | | [pbdNCDF4](http://cran.r-project.org/package=pbdNCDF4) | Parallel NetCDF version 4. ## Profiling | Package | Description | |---------|------------- | | [pbdPROF](href="http://cran.r-project.org/package=pbdPROF) | MPI profiling tools. | # Installation Instructions Below are sets of pbdR installation instructions for each major OS platform (or all three). These instructions include information for installing MPI and R in addition to installing the pbdR packages. If you have questions about installing a particular pbdR package, then you may need to consult with that package's vignette, which you can find on the [download page](download.html). We also maintain a Linux virtual machine (VM) with all of the pbdR packages installed and set up. You can find information about it [here](http://thirteen-01.stat.iastate.edu/snoweye/pbdr/?item=vm).
