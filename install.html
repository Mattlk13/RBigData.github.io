Allocation

All of our packages require an installation of an MPI library on top of an R installation that has been set up for package developers. It may be possible for us to manage these installation headaches for you, and even tailor our development to help your research.

If you are doing research at a United States institution, then please consider getting an allocation with XSEDE resources.


Installation Instructions

Below are sets of pbdR installation instructions for each major OS platform (or all three). These instructions include information for installing MPI and R in addition to installing the pbdR packages.

If you have questions about installing a particular pbdR package, then you may need to consult with that package's vignette, which you can find on the download page.

We also maintain a Linux virtual machine (VM) with all of the pbdR packages installed and set up. You can find information about it here. This is especially encouraged for Windows users, for whom the general installation process is non-trivial.
